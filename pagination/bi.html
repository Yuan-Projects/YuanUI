用法

引入所需的文件

引入JS

<script src="PATH_TO_FILE/jquery.pagination.js"></script>

引入CSS

<link rel="stylesheet" href="PATH_TO_FILE/jquery.pagination.css" />

确保CSS文件中指向图片的路径是正确的。

基础用法

在HTML页面中增加一个 div 元素，为它设置一个id属性，确保它在页面中是唯一的。

<div id="pagination"></div>

初始化分页插件，传递一个配置对象字面量到 pagination 方法。更多的可配置参数见配置参数列表。

  $('#pagination').pagination({
    total: 0
  });

请调整下拉框的值观察变化

<div id="pagination6"></div> 
  $('#pagination6').pagination({

得到插件的配置参数

在分页插件初始化之后，可以查询它的配置参数，通过 options 方法完成。点击下方的按钮将获得当前的总记录数。

<div id="pagination8"></div> 
  $('#pagination8').pagination({
    total: 100,
    pageList: [10, 30, 50, 100],
    pageSize: 30,
    pageNumber: 2
  });

  $('#pagination8-btn').on('click', function() {
    var option = $('#pagination8').pagination('options');
    alert('总记录数为：'+option.total);
    return false;
  });

示例：

刷新分页插件

分页插件初始化之后，可以刷新插件。点击下方按钮将会把总记录条数设置为200，当前页码数修改为第3页。

<div id="pagination9"></div> 
  $('#pagination9').pagination({
    total: 100,
    pageList: [10, 30, 50, 100],
    pageSize: 30,
    pageNumber: 2
  });
  $('#pagination9-btn').on('click', function() {
    var option = $('#pagination9').pagination('refresh', {
      total: 200,
      pageNumber: 3
    });
    return false;
  });

示例：

使用程序实现页码跳转

分页插件初始化之后，也可以用JS代码实现页码跳转，点击下方按钮，当前页码数跳转到第2页。

<div id="pagination10"></div> 
  $('#pagination10').pagination({
    total: 100,
    pageList: [10, 30, 50, 100],
    pageSize: 30,
    pageNumber: 1
  });
  $('#pagination10-btn').on('click', function() {
    var option = $('#pagination10').pagination('select', 2);
    return false;
  });

配置参数列表

属性名 类型 默认值 含义
total Integer 0 总记录数
pageList Array [10, 20, 30, 50] 一个数组，用于生成一个下拉框，控制每页显示的记录数。
pageSize Integer 10 每页的显示记录数
pageNumber Integer 1 当前页码数
onChangePageSize Function function(pageSize) {} 当每页显示的条数发生变化时的回调函数。
onSelectPage Function function(pageNumber, pageSize) {} 当页码数发生变化时的回调函数。

方法列表

方法名字 说明
options 查询插件设置配置参数
refresh 刷新分页插件
select 选择指定的页面数
